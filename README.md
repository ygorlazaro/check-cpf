# Check CPF

Ferramenta para checar se um CPF é válido.

## Como usar

Basta instalar com `npm i check-cpf` ou `yarn add check-cpf`.

O uso é bem direto:

```js
const checkCpf = require('check-cpf')

console.log(checkCpf('00000000000'))
```
